# GasNet: An AI-Driven Decision Support and Point-of-Sale System for CJG LPG Trading

## Project Overview
**GasNet** is an AI-Driven Decision Support and Point-of-Sale (POS) System developed for **CJG LPG Trading**. The project aims to improve business operations by automating sales recording, synchronizing data across multiple branches, and providing AI-based analytics for better decision-making.

This capstone project is divided into three main objectives, each assigned to a specific team member.

# Objective 2
## Seamless Integration and Synchronization Across Multiple Devices

**Owner:** Mendoza Iladio T.  
**Objective #:** 2

### Objective Title
To provide seamless integration and synchronization across multiple devices used in different branches.

### Objective Description
This objective focuses on enabling seamless integration and synchronization across multiple devices used in different CJG LPG Trading branches. It ensures that **sales, inventory, and operational data remain accurate and consistent** throughout the organization.

The goal is to support **real-time communication between POS tablets and the central server**, allowing data to be automatically shared and updated across all connected devices. This helps prevent discrepancies between branches, improves data reliability, and supports efficient business operations.

This objective also supports the other project objectives by:
- synchronizing sales records from the POS module,
- updating inventory information across devices,
- supplying accurate data for reporting, monitoring, and decision-making features.
